## 最终方案：HTML no-cache + 移除版本轮询（2026-06-29 最终版）
- 用户反馈静默刷新"用着用着会直接刷新"体验仍不好，决定改为行业标准方案B
- **最终方案：HTML no-cache，静态资源长缓存，完全移除版本轮询**
- Nginx: HTML 文件 `location ~* \.html$` 从 `expires 1h` 改为 `expires -1`，Cache-Control 改为 `no-cache, must-revalidate, proxy-revalidate`
- index.html: 移除所有 version.json 轮询 JS 代码（约30行）
- server.js: 移除 `GET /version.json` 和 `POST /api/bump-version` 接口
- deploy_all.py: 修正 Nginx 配置路径为 `/etc/nginx/sites-available/zyuit`
- 部署后验证：`curl -sI http://146.56.231.87` 返回 `Cache-Control: no-cache` ✅
